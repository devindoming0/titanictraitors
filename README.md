# TITANIC TRAITORS

A social deduction game for 3–8 players, designed for a 6-day cruise.
Inspired by James Cameron's *Titanic* and the music of Celine Dion.

---

## Files

| File | Contents |
|------|----------|
| `rulebook.md` | Full game rules — read this first |
| `characters.md` | All 8 characters with descriptions and optional abilities |
| `role-cards.md` | Printable role cards, kill sheets, and behavior cards |
| `day-by-day-guide.md` | Host schedule across 6 cruise days |

---

## Quick Start

1. Read `rulebook.md`
2. Print and cut cards from `role-cards.md`
3. Assign characters from `characters.md`
4. Follow the host schedule in `day-by-day-guide.md`

## Optional Rules

- **Suspicious Behavior** — Traitors must perform a quirky behavior daily
- **Character Abilities** — Each character has a one-use special power
- **Ghost Whisper** — Murdered players can share one hint before midnight
- **Locked Room** — One player per night can be immune from murder
